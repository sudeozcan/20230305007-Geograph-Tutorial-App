\# GEOAPP - Geography Tutorial App



GEOAPP is an interactive mobile learning application built with Android Studio. It is designed to help users explore geography through detailed country profiles and test their knowledge with integrated quiz modules.



\##  Key Features

\* \*\*Country Profiles:\*\* Access detailed geographic, demographic, and cultural information about various countries.

\* \*\*Interactive Quiz Module:\*\* Test your geography knowledge with dynamic questions and track your progress.

\* \*\*Offline Access:\*\* Powered by a local SQLite database, allowing users to learn anytime, anywhere.



\##  Technologies \& Tools

\* \*\*Platform:\*\* Android

\* \*\*Language:\*\* Java

\* \*\*Database:\*\* SQLite

\* \*\*IDE:\*\* Android Studio
